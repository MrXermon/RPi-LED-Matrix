### Cronjobs
    @reboot         screen -AmdS LED-Daemon /opt/led/start.sh
    * * * * *       cd /opt/led/; /usr/bin/python2.7 cron_telegram.py
    * * * * *       cd /opt/led/; /usr/bin/python2.7 cron_twitter.py
